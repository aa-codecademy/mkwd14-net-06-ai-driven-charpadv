# 📚 C# Advanced – Extra Resources

This document contains **optional practice materials** and **useful resources** for students who want to:
- Strengthen their C# language fundamentals
- Master OOP and advanced language features
- Practice problem-solving in C#
- Learn from high-quality external resources


## 🌐 General C# Resources

Use these when you want to **learn, read, or look things up**.

  Official C# documentation (Microsoft Learn)

- 🔗 https://learn.microsoft.com/en-us/dotnet/csharp/tour-of-csharp/  
  A guided tour of the C# language

- 🔗 https://dotnetfiddle.net/

- 🔗 https://learn.microsoft.com/en-us/dotnet/csharp/linq/  
  Official LINQ documentation with examples

- 🔗 https://linqpad.net/  
  Interactive tool for testing and exploring LINQ queries

- 🔗 https://learn.microsoft.com/en-us/dotnet/standard/collections/
