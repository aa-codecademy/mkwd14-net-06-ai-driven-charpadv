## 🤖 Using AI in Homework

You are allowed to use AI tools (ChatGPT, Copilot, etc.) while working on this homework.

---

### 🧠 How to use AI

- Use AI to understand generics and reuse of code  
- Ask for explanations, not full implementations  
- Break the problem into smaller parts  
- Use AI when stuck, not as a first step  
- Always understand the code you write  

---

### ⚙️ Suggested approach

1. Solve Task 1 (focus on generics and reuse)  
2. Review previous homework (Vehicle classes)  
3. Extend functionality without changing existing logic  
4. Test all methods  

---

### ❌ Avoid

- Copy/pasting full solutions  
- Rewriting previous homework instead of extending it  
- Asking AI to generate full code
